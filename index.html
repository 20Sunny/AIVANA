ABOUT AIVANA

Welcome to the AIVANA parent company repository! AIVANA is a technology company that serves as the parent organization for several capstone projects. This repository houses the main website for AIVANA, which has been developed using React.js. The website serves as a comprehensive resource, offering an overview of AIVANA as well as detailed information about its various projects.

AIVANA is a forward-thinking parent company that specializes in innovative projects and cutting-edge technologies. The primary objective of AIVANA is to develop and deliver solutions that leverage the power of artificial intelligence (AI) and machine learning (ML). With a focus on pushing the boundaries of what is possible, AIVANA strives to create impactful applications that bring significant value to its users.

AIVLOG - A Blog website

AIVLOG - A Blog website

AIVLOG is a blog application developed under the AIVANA parent company. AIVLOG allows users to upload blogs categorized into five different categories. Users who upload blogs have full CRUD (Create, Read, Update, Delete) operations on their own blogs, while other users can read the blogs, leave comments, and delete their own comments. Additionally, users can set a thumbnail for their blogs in various image formats such as .jpg, .png, .gif, or .svg.

A dynamic MERN stack blog application. Empower bloggers to share, engage, and create with seamless CRUD operations, intuitive UI, and captivating thumbnails.
